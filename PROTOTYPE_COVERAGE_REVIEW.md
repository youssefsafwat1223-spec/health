## المغطى في البروتوتايب

- إجمالي الشاشات الحالية: 64 شاشة داخل Prototype واحد.
- Onboarding: Splash، اللغة، نوع الحساب، اختيار نوع مقدم الخدمة، تسجيل المريض، وتسجيلات منفصلة للطبيب والصيدلية والمختبر والأشعة.
- Patient App: الرئيسية، كل الخدمات، البحث الذكي، الباركود، البحث بالأعراض، تفاصيل الدواء، الأطباء، بروفايل الطبيب، الحجز، تأكيد الحجز، الصيدلية، إرسال وصفة، المختبر، مساعد المختبر، الأشعة، مساعد الأشعة، الخريطة، صحتي، أدوات الصحة، أدويتي، مقارنة المنتجات، المحادثات، الشات، السلة، الدفع، الأكواد والخصومات، الإشعارات، المفضلة، حسابي.
- Provider App: لوحة الصيدلية، لوحات الطبيب والمختبر والأشعة، الطلبات، المنتجات، إضافة/تعديل منتج، إضافة/تعديل خدمة، صلاحيات الكادر، معاينة البروفايل، إعدادات المزود، RX Group، مندوب شركة، الاشتراكات.
- Admin: Dashboard مختصر مع Sidebar، إدارة المستخدمين، مراجعة المزودين، الاشتراكات والمدفوعات، الإعلانات والشكاوى.
- UI system: RTL، Cairo، dark/light toggle، phone frame، sidebar للشاشات، bottom nav للمريض والمقدم.
- قواعد الصيدلية الخاصة: RX Group، MR Group، حساب الجرعة، مزودون قريبون.
- 14 أيقونة في الرئيسية.
- محتوى عراقي: بغداد، البصرة، أربيل، دينار عراقي.

## مغطى بشكل مختصر وليس تفصيلي كامل

- ربط POS موجود كإشارة واضحة في تسجيل الصيدلية وإعدادات المزود وخطة الاشتراك الذهبية، لكنه ليس Flow تشغيلي كامل داخل البروتوتايب.
- التحقق الإداري متعدد الخطوات موجود كشاشة مراجعة وPending Verification، لكنه ليس Workflow كامل لكل نوع وثيقة.
- Error states وOffline states مذكورة وممثلة داخل الإعدادات وبعض الشاشات، لكنها ليست شاشة مستقلة لكل flow لأنها عادة تظهر داخل السياق نفسه.
- لوحة الإدارة تعرض أهم الوحدات للعميل، لكنها ليست Dashboard تشغيلية كاملة بكل التقارير والفلاتر.

## الخلاصة

البروتوتايب مناسب كعرض Client-facing شامل. بعد المراجعة الثانية، الشاشات التي كانت تبدو ناقصة مثل الاشتراكات، الباركود، الذكاء/البحث الآمن، مساعد المختبر، مساعد الأشعة، الخريطة، أدوات الصحة، أدويتي، مقارنة المنتجات، الدفع، والأكواد أصبحت موجودة. المتبقي الحقيقي قبل التنفيذ ليس شاشات عرض للعميل، بل تحويل بعض البنود التشغيلية إلى flows إنتاجية كاملة مثل POS، مراجعة الوثائق بالتفصيل، تقارير الإدارة، وحالات الخطأ داخل كل عملية.
